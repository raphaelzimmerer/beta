GPT100 - Gas Pressure Sensor

  1. Products
  2. Product Details

Product Specification

  • Process Connection: G1/4" made of stainless steel AISI 316L
  • Enclosure: Made of weather-resistant aluminum 6082-T6
  • Screw Cap: Made of UV-stable high-performance plastic
  • Design: Sustainable due to modular structure and robust design
  • Components: All parts replaceable (e.g. battery, electronics)
  • Operation: Energy-efficient ultra-low power operation 24/7
  • Component Quality: Use of high-quality components
  • Sensor: Precise, long-term stable premium gas pressure sensor
  • Battery: Durable Li-SOCl₂ brand battery with LIC for pulse operation
  • Electronics: “Made in Germany” (hardware and software)
  • Commissioning: Integrated on/off switch for easy start-up
  • Status Indicator: Status and signal strength display via integrated LED

Technical Data

  • Data Transmission: Transmits gas pressure, temperature, battery status, and more
  • Pressure Range: 0...10 bar absolute
  • Total Error Band: ± 0.70 % FS
  • Non-Linearity (BFSL): ± 0.15 % FS
  • Long-Term Stability: ≤ ± 0.20 % FS (under reference conditions)
  • Temperature Measurement Accuracy: ± 2 °C typical
  • Ambient Temperature*: -20...40 °C
  • Battery Life: > 10 years @ 4 samples per day

LoRa Parameters

  • • Protocol: Class A, OTAA, FUOTA
  • • Frequency: LoRa 868 MHz EU, integrated antenna
  • • Data Rate: Configurable as adaptive or static
  • • Microcontroller: Modern, energy-efficient microcontroller
  • • Security: Secure key management hardware
  • • LoRa Version: 1.0.4

Optional Features

  • • Various process connections
  • • Custom process adapters (e.g. T-piece)
  • • Customized firmware
  • • Other measurement ranges and media (gases, liquids)
  • • External LoRa antenna
  • • LoRa Class B implementation
  • • Support with planning, installation, integration, and service

Scope of Delivery

  • • System: LoRa gas monitor with process connection
  • • Power Supply: Powered by integrated batteries
  • • Documentation: Operating manual
  • • Software: JavaScript payload formatter
  • • Integration: Integration guide

Dimensions

  • • Dimensions (L×D): 128×50 [mm]
  • • Weight: 0.3 [kg]
